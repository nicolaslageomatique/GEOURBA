Le format VRT

23/02/2026 / Jean

Qu'est-ce qu'un fichier VRT ?

Les fichiers VRT appartiennent principalement à GDAL de GDAL Developers. Un fichier VRT est un monde virtuel 3D enregistré dans le OGR Virtual Format utilisé par le GDAL bibliothèque de traduction pour les ensembles de données géospatiales raster et vectorielles. Un tel monde virtuel 3D est enregistré en tant que jeu de données raster ou vecteur. Un fichier VRT enregistré en tant que jeu de données vectorielles est écrit en Extensible Markup Language (XML ). Il contient une collection de textures, de sources, de mappages et de calques qui composent une scène virtuelle. Un autre type de fichier VRT est une définition de profil utilisée par Civil 3D, logiciel de conception et de documentation de structures civiles.

L'intéret du VRT c'est qu'il permet de visualiser une vue spécifique d'une donnée.
